# Changelog

## [2.0.0] - 2026-02-07

### Added
- Syntax highlighting for Semi Flow Language (.sfl, .sfli, .sflc)
- Code snippets for all scheduler types (MSC, WSC, RSC, Station)
- Snippets for transactions, pub/sub, state machines
- Language configuration with bracket matching
- Code folding support

### Supported Keywords
- Schedulers: MASTER_SCHEDULER, WAFER_SCHEDULER, ROBOT_SCHEDULER, STATION
- Blocks: SCHEDULE, CONFIG, VERIFY, STATE_MACHINE
- Rules: APPLY_RULE, FORMULA
- Layers: L1, L2, L3, L4
- Messaging: publish, subscribe, transaction
